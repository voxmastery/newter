# Changelog

## 0.1.0 — Initial Release

- Syntax highlighting for `.newt` files via TextMate grammar
- LSP integration for real-time diagnostics (requires `newter-lsp`)
- **Newt: Run current file** — compile and run the active `.newt` file
- **Newt: Check current file** — type-check / validate without running
- **Newt: Export to HTML** — export the current file to HTML
- **Newt: Open Canvas IDE (live preview)** — start a live-preview dev server
- Configurable compiler and LSP binary paths
